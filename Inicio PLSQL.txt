CLOB - ARMAZENAMENTO DE ARQUIVO EM TEXTO ( O BANCO ABRE E EXIBE EM UMA LINHA ) 
BFILE - LINK PARA O LOCAL ONDE ESTA ARMAZENADO
RAW - ARMAZENA DADOS BINARIOS , ARMAZENA 34K
LONG RAW - ARMAZENA DADOS BINARIOS ATE 2GB
LONG - ARMAZENA TEXTO ATE 2GB
NCHAR - ARMAZENA DADOS SOLICITADOS SEM ALTERAÇÃO
